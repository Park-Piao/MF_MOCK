1. HTML boilerplate:

In blank index.html
Type ! ("Shift" + 1)
Press "Tab" button 

2. Commenting:

Windows/Linux: Ctrl + /
Mac: Cmd + /

3. Emmet built-in for HTML shortcuts:

Type this → Press Tab → Get this:

div → <div></div>
p → <p></p>
h1 → <h1></h1>
script → <script></script>





VS Code:

Ctrl/Cmd + / → comment
Ctrl/Cmd + D → select next occurrence
Alt + Up/Down → move line up/down
Shift + Alt + Down → duplicate line
